Bienvenido a mi sito Web

Trabajo Práctico Integrador

Este sitio web fue desarrollado como parte del Trabajo Práctico Integrador de la materia. El proyecto busca demostrar el uso de HTML, CSS y JavaScript para crear una experiencia web moderna, funcional y agradable para el usuario.

En esta página de inicio se presenta una introducción general sobre el tema principal del sitio: la importancia de que las páginas web no pidan la contraseña una y otra vez. A través del diseño y la programación, se busca reflejar cómo una buena experiencia de usuario también implica equilibrio entre seguridad y comodidad.

La sección de "Artículo" profundiza en esta idea, analizando por qué las páginas que piden la contraseña en exceso pueden generar frustración, y proponiendo soluciones modernas como la autenticación persistente y el inicio de sesión seguro sin molestias innecesarias.

Por otro lado, en la página de "Contacto" se encuentra un formulario validado con JavaScript, donde se aplican expresiones regulares para comprobar datos como el nombre, el correo y el teléfono, simulando un entorno web profesional y seguro.

Todo el sitio fue desarrollado utilizando únicamente HTML, CSS y JavaScript puro
