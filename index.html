Bienvenido a mi sito Web

Trabajo Práctico Integrador

Este sitio web fue desarrollado como parte del Trabajo Práctico Integrador de la materia. Está compuesto por tres páginas principales conectadas mediante un menú de navegación utilizando Flexbox.

En la página de inicio se encuentra esta información general junto a un carrusel de imágenes programado en JavaScript, utilizando arrays y botones para avanzar y retroceder de forma circular.

En la sección "Artículo", se presenta una disposición de contenido en múltiples columnas utilizando Flexbox, simulando una galería de productos o servicios.

Por último, en la página de "Contacto" se encuentra un formulario con validaciones personalizadas en JavaScript. Se verifican campos obligatorios, formato de email y teléfono mediante expresiones regulares, y se muestran errores o los datos ingresados directamente en la página sin usar alertas nativas.

Todo el sitio fue desarrollado con HTML, CSS y JavaScript sin utilizar frameworks ni bibliotecas externas, y se encuentra subido en GitHub con despliegue a través de GitHub Pages.
